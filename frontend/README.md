# Frontend Administrativo

Aplicacion Vue 3 con Vite para el modulo administrativo.

## Estructura

- `src/pages`: pantallas principales por modulo.
- `src/layouts`: estructuras globales como sidebar y shell.
- `src/components/common`: componentes reutilizables generales.
- `src/components/drivers`: componentes especificos del modulo de choferes.
- `src/composables`: estado y casos de uso reutilizables.
- `src/constants`: constantes de navegacion y modulos.
- `src/services`: clientes HTTP/API.
- `src/utils`: funciones puras compartidas.

## Comandos

```bash
npm install
npm run dev
npm run build
```
